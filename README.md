# articleous

### dummy tool supposed to assist you in learning article grammar

Demo with non-persistent redis memory is hosted [here](https://articleous.herokuapp.com)

## Run locally

### Prerequisites
* node
* redis

### Setup

0. Clone this repo
1. `redis-server`
2. `npm run dev`

[Sapper](https://sapper.svelte.dev/) + [Svelte](https://svelte.dev/)
